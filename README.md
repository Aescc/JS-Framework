# JS-Framework
Some simple things to help out in JavaScript.
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Commands Here:
	Circle:
		-Draws a circle.
		-Parameters: x, y, size, color
	FindAngle:
		-Finds the angle between two points.
		-Parameters: x1, y1, x2, y2
	FindDistance:
		-Finds the distance between two points.
		-Parameters: x1, y1, x2, y2
