# JS-Framework
Some simple things to help out in JavaScript.
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Commands Here:
	Circle:
		-Draws a circle.
		-Parameters: x, y, size, color.
	FindAngle:
		-Returns the angle between two points.
		-Parameters: x1, y1, x2, y2.
	FindDistance:
		-Returns the distance between two points.
		-Parameters: x1, y1, x2, y2.
	HitTest:
		-Tests if two user defined rectangles are colliding.
		-Parameters: x1, y1, width1, height1, x2, y2, width2, height2.
	Line:
		-Draws a line.
		-Parameters: x1, y1, x2, y2, color, width.
	Random:
		-Returns a random number with user specified bounds.
		-Parameters: minimum, maximum.
	Rect:
		-Draws a rectangle.
		-Parameters: x, y, width, height, color, opacity(optional).
	Text:
		-Writes some text.
		-Parameters: x, y, message, color, font style(optional).
